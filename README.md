# Spiritus_Invictus

PREREQUISITES: 
- Operating System: Windows 10 or Later.
- Processor: 2 GHz.
- Memory: 1 GB RAM.
- Graphics Card: Minimum Resolution 1280x720px.
- Hard disk: 1 GB of Available Space.
- Can be Played with a Keyboard & Mouse, but a Controller is Recommended.
